Tea Kettle Chat Box

Provided are some phrases you can ask the chat box:

  • Turn on/off the tea kettle.
  • Options for tea types?
  • Choosing a tea.
  • Please make the tea temperature at... Celsius/Fahrenheit.
  • Turn up/down the tea temperature at ... Celsius/Fahrenheit.
  • Is the tea boiling?
  • Is the tea done?

    • After test 1:

    What worked well in user test 1:

    • User used various versions of the provided questions.
    • User preferred simpler responses like "kettle on/off."
    • User liked the feedback received when selecting a specific tea.

    Issues identified in user test 1:

    • Some phrases didn't trigger correct responses, leading to empty responses.
    • Missing a completion state before turning off the tea kettle in the dialog tree.
    • Spelling and grammatical errors were found.
    • The chat box location is not easy to read on the screen.

    Improvements made after user test 1:

    • Added more training phrases within the intents.
    • Added phrases/intents for: "Is the tea done yet?"
    • Revised wording in responses for clarity.

    After test 2:

    What worked well in user test 2:

    • User used different variations of the provided questions.
    • User provided more detailed responses, including filler words like "please."
    • User appreciated the ease of use of the chat box.

    Issues identified in user test 2:

    • Some phrases still didn't trigger correct responses, resulting in empty responses.
    • Added more phrases for temperature changes.
    • Spelling and grammatical errors were still present.

    Improvements made after user test 2:

    • Created/Added more training phrases within the intents.
    • Added phrases/intents for: "Please make the tea temperature at... Celsius/Fahrenheit."
    • Added phrases/intents for: "Increase/decrease temperature."
    • Revised and simplified responses for clarity.

    After test 3:

    What worked well in user test 3:

    • User used different variations of the provided questions.
    • User provided shorter responses.
    • User still experienced some empty responses.
    • User appreciated the flexibility of the chat bot, which was intuitive with small misspellings.

    Issues identified in user test 3:

    • Some phrases still didn't trigger correct responses, leading to empty responses.
    • User asked if the tea was in the boiling stage.
    • More spelling and grammatical errors were found in responses.

    Improvements made after user test 3:

    • Provided additional training phrases within the intents.
    • Added intent for the stage of tea kettle.
    • Added phrases/intents for: "Is the tea boiling?"
    • Fixed spelling errors in responses.

    About Section

    This web application contains a chatbox using Google dialogflow for a basic smart tea kettle appliance. You can ask it simple questions like turning the kettle on or off, selecting tea options, adjusting the temperature, and checking if the tea is boiling or if the tea is ready.
